## Context
- Source discussion:
  - 基于 `docs/02-strategy/discussions/trading-strategy-bg.md` 中 `3.1 评分维度定义 / 维度一：成交量分析` 的讨论。
  - 当前原始公式包含 `历史最大成交量` 归一化口径，存在极值敏感与潜在前视偏差风险。
- Problem or opportunity:
  - 量能评分方向正确（放量水平 + 异常放量 + 持续趋势），但实现口径未收敛，容易出现跨周期不可比、单项主导、极值失真。
- Why not now:
  - 当前优先完成指标层能力与策略分析，评分系统尚未进入实现阶段。

## Desired Outcome
- 在评分系统开发前，冻结“成交量评分 v1”可实现口径，满足：
  - 无前视偏差；
  - 跨时间周期可解释；
  - 各子项同量纲、可加总；
  - 能直接接入后续 `signal_engine / scoring`。

## Proposed Direction
- v1 候选定义：
  - `ratio`: `volume / sma(volume, 20) - 1`（当前量相对常态量）
  - `spike`: `zscore(volume, lookback=60)`（替代“最近5日峰值 / 历史最大成交量”）
  - `trend`: `slope(log(sma(volume, 20)), window=10)`（量能持续性）
  - 三项先各自 clip + normalize，再按权重加总为 `volume_score`
- Recommended:
  - 先采用上面 v1 口径并保留参数化窗口（20/60/10），后续可基于回测再调参。

## Trigger Condition
- Start implementation when:

  - 或需要把五维评分接入回测信号流水线。
